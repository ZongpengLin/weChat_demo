# weChat_demo
基于SpringBoot+MyBatis开发的迷你微信小程序（后端部分）

项目介绍：
- 以商品信息的增删改查为例，依据业务模块配合UT实现后端完整的增删改查功能，最后来到微信小程序开发部分，实现功能显示的前端渲染。

商品表信息：
- 存储在 MySQL 中，主要包括商品的三个属性字段： id(编号)、name(名称)、priority(优先级)；
